About

If you go looking around for open-source microblogging platforms on the web these days, one thing stands out: their age. Most of them were developed around the time Twitter gained prominance, and thus they reflect the technology of that time. I wanted a more accessible, more hackable Twitter clone that was nonetheless easy to set up. And so Roar was born. Built on Meteor.js, it can be booted up in minutes, as well as easily tweaked.

Features

All the core features of Twitter:

  • Hashtags

  • Tag Users

  • Direct messaging

  • Comment system

  • Elastic Search

  • Notifications

  • Reposting

  • Post Photos

  • Favoriting
